# **App Name**: DJ Pad Controller

## Core Features:

- Customizable 5x5 Pad Grid: A 5x5 grid of buttons (pads) each associated with a sound sample.
- Sound Triggering: Trigger audio playback on pad press, supporting simultaneous audio playback.
- Loop Toggle: Toggle loop playback for each individual pad.
- Pitch Adjustment: Individual pitch sliders for pads to adjust playback speed/tone using the Web Audio API.
- Bass EQ Adjustment: Individual bass EQ sliders to filter low frequencies, using the Web Audio API.
- Global Master Volume: Master volume control slider for overall audio output.
- Customizable Pad Colors: Allow users to customize the colors of each pad through a color picker.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to represent music and creativity.
- Background color: Dark gray (#303030) to provide contrast and focus on the pads.
- Accent color: Electric green (#7CFC00) for active pad indication and controls.
- Body and headline font: 'Inter' sans-serif, for a modern and clean interface.
- Responsive grid layout ensuring the 5x5 pad grid adapts to different screen sizes.
- Simple, geometric icons for loop, play, and pause controls.
- Subtle feedback animation (e.g., a brief highlight) when a pad is pressed.
